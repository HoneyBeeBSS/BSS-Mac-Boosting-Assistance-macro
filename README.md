**_Honey Macro_**

**Boosting Assistance Macro – made by Honey Bee**

A lightweight macOS macro designed to automate item usage while boosting in Bee Swarm.
The macro allows you to schedule automatic key presses for slots 1–7 with custom intervals, so you can focus on gameplay while the macro handles the repetitive actions.

**Features**
- Automatic key presses for slots 1–7
- Customizable interval per slot
- Start/Stop buttons inside the app
- Global hotkey (Cmd + Shift + B) to toggle the macro even in fullscreen apps
- Visual ON/OFF indicator
- Lightweight and easy to use


**How to Use**
1. Download and unzip the app.
2. Open the app for the first time: macOS will ask for permissions. Follow these steps:
  a. Go to System Settings → Privacy & Security → Accessibility, and enable Honey Macro.
  b. Go to System Settings → Privacy & Security → Input Monitoring, and enable Honey Macro.
3. Use the toggles to enable the slots you want the macro to control.
4. Set the interval (in seconds) for each slot.
5. Press Start Macro to begin automation.
6. Press Stop Macro to stop automation.
7. Use Cmd + Shift + B as a global hotkey to start/stop the macro anytime.

**Notes**
- First-time users might see a macOS warning about an unidentified developer. Right-click → Open → Open again to bypass.
- Make sure the app is running and permissions are enabled; otherwise, the macro won’t function.
